# Кода-стайл проекта

## Общие правила

- Только ES-модули (`import/export`).
- Строки — одинарные кавычки.
- Отступы — 2 пробела.
- Максимальная длина строки — 100 символов.

## JavaScript

- Каждая функция выполняет одно действие.
- Не использовать глобальные переменные.
- Все API-вызовы только через services.
- Компоненты UI — чистые, без бизнес-логики.

## CSS

- БЭМ по необходимости.
- Никаких inline-стилей.
- Цвета через CSS-переменные.
- Один компонент = один CSS-файл.
